# 温雅的农药查询工具

## 简介

查询王者荣耀账号的近期战绩/实时上榜战力/账号资料等功能

## 使用方式

功能          → 功能总览

战绩 [标识]   → 最近对局战绩（需配置应天API令牌）[前往注册](https://api.t1qq.com)

资料 [标识]   → 角色资料说明（需配置应天API令牌）[前往注册](https://api.t1qq.com)

战力 英雄 大区(参数"aqq", "awx", "iqq", "iwx",""或"all"输出四区) (需配置柠柚API令牌) [前往注册](https://api.nycnm.cn)

查看       → 显示全部映射

添加 营地ID 名称

修改 营地ID 名称

删除 营地ID

查询 [标识] → 显示[标识]映射

[标识] = 自定义角色名 或 营地ID

如果开启了前缀，需要在所有指令前面加上设定的前缀。

## 优化内容

以纯文本形式输出

重构精简工程库所有代码文本结构以及内容

简化输出流程以及信息流
